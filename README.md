# Setup

[readthedocs getting started](https://docs.readthedocs.io/en/stable/intro/getting-started-with-sphinx.html)

	pip install sphinx --user
	
	mkdir project_name
	cd project_name
	git init
	
	mkdir docs
	cd docs
	sphinx-quickstart
	make html
